실시간 용해도 곡선 KNO₃ CuSO₄ 물 100g 기준

실험 순서

1
준비 용질 병 클릭 → 숟가락에 담겨 커서를 따라옴 → 비커를 클릭하면 녹음
2
냉각 비커를 얼음물 수조로 드래그 → 1초당 2°C씩 자동 냉각. 그래프의 점이 좌측으로 이동하며 곡선을 만나는 순간 결정 석출!
3
여과 비커를 거름 장치로 드래그 → 결정이 거름종이 위에 분리됨

석출량 계산

석출량(g) = 처음 녹인 양 − 냉각 온도 용해도 예) KNO₃ 30g, 0°C 냉각 → 30 − 13(0°C 용해도) = 17g 석출
💡 왜 KNO₃ 가 먼저?
KNO₃ 곡선은 기울기가 급해서 온도가 조금만 내려가도 용해도가 크게 줄어들어 결정이 많이 생깁니다. CuSO₄는 곡선이 완만해서 냉각해도 거의 석출되지 않아요.
📚 재결정(Recrystallization)
온도에 따른 용해도 차이를 이용해 혼합물에서 순수한 고체를 분리하는 방법. 여러 번 반복할수록 순도가 높아집니다.
⚗️ 분별 결정
재결정 과정을 여러 번 반복하여 혼합물의 각 성분을 분리하는 방법입니다. 설탕 공장에서 사탕수수로부터 설탕을 분리할 때 이 방법을 씁니다.
